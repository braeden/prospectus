# Prospectus

### Express (backend)
```
npm install
npm start
```

### React (frontend)
```
cd frontend
npm install
npm start
```


## .env
```
DB_HOST=1.1.1.1
DB_DATABASE=testDB
DB_USER=root
DB_PASS=password_here
CLIENT_ID=.....apps.googleusercontent.com
TOKEN_SECRET=JWT_SECRET_KEY
```

## env_variables.yaml (for GCP deploy)
```
env_variables:
  DB_HOST: ''
  DB_DATABASE: 'Prospectus'
  DB_USER: ''
  DB_PASS: ''
  CLIENT_ID: '.apps.googleusercontent.com'
  TOKEN_SECRET: 'JWT SECRET KEY'
```
